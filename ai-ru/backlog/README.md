# backlog/ — запланированные фичи

Файлы: `BFN_SHORT_NAME.md` (BF = Backlog Feature).

Отличие от `features/`: описывается **будущая** фича, которую ещё не реализовали.

Статусы в заголовке:
- 📋 **Planned** — согласована, ждёт очереди
- 🟡 **In progress** — разрабатывается (после мержа переехать в `features/`)
- ❄️ **Frozen** — отложена на неопределённый срок

Шаблон — см. [../features/README.md](../features/README.md), но разделы "Реализация" → "План реализации", "Ограничения" → "Открытые вопросы".
