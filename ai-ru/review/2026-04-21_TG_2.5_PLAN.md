### Step 2 — impl

```java
public SceneOutput renderNotification(TextMessage message) {
    UUID notifId = message.getNotificationId();
    Notification n = notificationService.findByIdOrThrowNotFound(notifId);

    String emoji = resolveEmoji(n.getReasonId());
    String qrName = MessageUtils.escapeHtml(n.getQr() != null && n.getQr().getName() != null
        ? n.getQr().getName() : "Метка");
    String qrMeta = n.getQr() != null && n.getQr().getSeqNumber() != null
        ? " · №" + n.getQr().getSeqNumber() : "";

    StringBuilder body = new StringBuilder();
    body.append(emoji).append(" <b>").append(qrName).append("</b>").append(qrMeta).append('\n');
    if (n.getReason() != null && n.getReason().getDescription() != null) {
        body.append(MessageUtils.escapeHtml(n.getReason().getDescription())).append('\n');
    }
    String userText = n.getText();
    if (userText != null && !userText.isBlank()
            && (n.getReason() == null || !userText.equals(n.getReason().getDescription()))) {
        body.append('\n').append('«').append(MessageUtils.escapeHtml(userText)).append('»').append('\n');
    }
    if (n.getCreatedDate() != null) {
        body.append('\n').append("<i>")
            .append(formatDateRu(n.getCreatedDate()))
            .append("</i>");
    }

    InlineKeyboardMarkup kb = new InlineKeyboardMarkup();
    List<List<InlineKeyboardButton>> rows = new ArrayList<>();
    List<InlineKeyboardButton> row = new ArrayList<>();
    if (notificationService.shouldShowMarkAsReadButton(notifId)) {
        row.add(btn(messages.get("tg.notif.card.mark_read"), "notif:read:" + notifId));
    }
    if (n.getQrId() != null) {
        row.add(btn(messages.get("tg.notif.card.to_qr"), "qr_details:open:" + n.getQrId()));
    }
    if (!row.isEmpty()) rows.add(row);
    kb.setKeyboard(rows);
    return SceneOutput.sendHtml(body.toString(), kb);
}

private String resolveEmoji(Long reasonId) {
    if (reasonId == null) return messages.get("tg.reason.emoji.default");
    try {
        return messages.get("tg.reason.emoji." + reasonId);
    } catch (Exception e) {
        return messages.get("tg.reason.emoji.default");
    }
}

// DateTimeFormatter — locale "ru", "d MMMM" + ", HH:mm"
```

**Дополнение**: `handle(CallbackData, ctx)` — сейчас там убирает клавиатуру при `notif:read:<id>`. Нужно чтобы после «Отмечено» также добавлялась строчка `<i>✓ Прочитано</i>` к тексту карточки. Либо упрощаем: просто убираем клавиатуру (текущее поведение сохраняется), а «Прочитано» — в следующий спек. **Выбираем упрощение** — не усложняем 2.5, минимальная дельта. В спеке выше упомянуто «в перспективе», а в 2.5 — без изменений клавиатура-removal flow.
